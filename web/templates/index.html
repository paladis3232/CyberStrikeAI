Bot Command Reference

The following commands can be sent in a conversation:

  • help — Show this help
  • list — List all conversation titles and IDs
  • switch <ID> — Switch to the specified conversation
  • new — Start a new conversation
  • Clear clear — Clear current context
  • current — Show current conversation ID and title
  • stop — Interrupt the current task
  • roles — List all available roles
  • role <name> — Switch to the specified role
  • delete <ID> — Delete the specified conversation
  • version — Show current version

Any text other than the commands above is forwarded to the AI for penetration testing / security analysis.
